# 🩸 RaktNest – Blood Donation Management System

## 📌 Project Overview
RaktNest is a modern blood donation management web application developed to connect blood donors, patients, and administrators on a single platform. The project helps users easily request blood, register as donors, and manage blood donation activities efficiently.

This project is built using modern frontend technologies with authentication and dashboard management features for both users and admins.

# 📂 Project Flow

## 🏠 1. Landing Page
- Users visit the homepage.
- Website introduces blood donation services.
- Navigation bar provides access to all sections.

---

## 🔐 2. Authentication System
- Users can register and login.
- Admin has separate authentication panel.
- Session management handles user access securely.

---

## 🩸 3. Blood Donation Process
- Donors submit donation details.
- Blood group and personal information are stored.
- Requests can be managed through dashboard.

---

## 📊 4. Dashboard Management

### 👤 User Dashboard
- View profile
- Manage requests
- Logout securely

### 🛠️ Admin Dashboard
- Monitor users
- Handle blood requests
- Manage donation records

---

## 📞 5. Contact & Support
- Users can submit queries through contact form.
- Helps improve communication between users and admin.
